hackbikeshareTO Project API

API Call Description
/api/bikeDuration?start=X&end=Y&weather=W Takes in the start bixi station, end bixi station, and weather and returns the estimated trip duration in minutes as a JSON object.
/api/walkDuration?start=X&end=Y Takes in the starting location, end location, and returns the estimated trip duration. This API call pulls information from Google Maps, and requires (ADDRESS OR COORDS?) given as input arguments.
